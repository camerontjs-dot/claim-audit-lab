## LinkedIn Featured Caption

Claim Audit Lab is a deterministic Python CLI that checks whether draft claims are supported by supplied evidence. It produces inspectable Markdown and JSON reports with claim IDs, evidence links, support labels, risk labels, rule flags, and limitations. The demo runs locally with fictional fixtures and requires no network access or API key.
